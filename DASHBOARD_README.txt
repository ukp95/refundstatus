# SabPaisa DJB Refund Dashboard

This package contains the full source code for your dashboard, including backend (FastAPI) and frontend (HTML/JS).

## How to use

1. **Backend**
   - Location: `backend/main.py`
   - Requirements: Python 3.8+, FastAPI, pandas, uvicorn, openpyxl, python-multipart
   - To run: `uvicorn backend.main:app --reload`

2. **Frontend**
   - Location: `frontend/index.html`
   - Open with VS Code Live Server or any static web server.
   - Connects to backend at `http://localhost:8000`.

## Files
- `backend/main.py`: FastAPI backend code
- `frontend/index.html`: Modern dashboard UI

## Backup
You can copy this folder anywhere and use it as a backup. All code is self-contained and ready to run.
